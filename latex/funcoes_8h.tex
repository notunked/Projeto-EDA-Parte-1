\doxysubsection{Detailed Description}
Declarações de funções auxiliares para a manipulação de antenas e locais nefastos. 

Este ficheiro contém as definições das funções responsáveis por\+:
\begin{DoxyItemize}
\item Carregar antenas e a matriz do mapa a partir de ficheiros.
\item Gerenciar a lista ligada de antenas, incluindo inserção e remoção.
\item Identificar e armazenar locais considerados nefastos com base na posição das antenas.
\item Limpar a memória alocada dinamicamente para as listas. 
\end{DoxyItemize}
