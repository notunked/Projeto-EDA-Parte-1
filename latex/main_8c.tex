\doxysubsection{Detailed Description}
Programa para manipular antenas e identificar locais com efeito nefasto. 

Este programa carrega um mapa de antenas a partir de um ficheiro, armazena as antenas numa lista ligada e determina as posições com efeito nefasto ao redor de cada antena. Ele também permite a remoção de antenas e listagem das informações no terminal.
